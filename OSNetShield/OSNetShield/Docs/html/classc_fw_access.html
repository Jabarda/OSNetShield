std::wstring cFwAccess::makeRuleName ( )

The method creates a name for a new rule Although it's possible to create rules with same names, the names must differ for proper rule deleting The naming is based on wstrings in vFwAddedRules from cFwAccess class, which contains the names of the previously added rules The name of the new rule looks like "OSNetShield" + the smallest available number std::string makeRuleName(std::vector<std::wstring> &vFwAddedRules);
